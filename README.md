🏢 회의실 예약 관리 프로그램 (Meeting Room Booking System)

일반 HTML, CSS, JavaScript(바닐라 JS) 로 구현한 회의실 예약 관리 프로그램입니다.
서버나 데이터베이스 없이도 동작할 수 있도록 LocalStorage를 활용하여 예약 정보를 저장합니다.

🔧 주요 기능
✅ 1. 회의실 목록 조회

사용자는 회의실 리스트를 확인할 수 있습니다.

각 회의실의 현재 예약 현황(예약자 이름, 예약 시간 등)을 실시간으로 표시합니다.

✅ 2. 회의실 예약

사용자 이름과 비밀번호를 입력하여 회의실을 예약할 수 있습니다.

예약 정보는 LocalStorage에 저장되어 브라우저를 새로고침해도 유지됩니다.

✅ 3. 예약 수정

예약 시 입력했던 이름 + 비밀번호가 일치하면 예약 내용을 수정할 수 있습니다.

✅ 4. 예약 취소

예약자 본인이 이름 + 비밀번호 인증을 통해 예약을 취소할 수 있습니다.

💾 데이터 저장 방식

데이터베이스(DB) 없이 동작하도록
브라우저 LocalStorage를 이용해 예약 정보를 저장/수정/삭제합니다.

다른 사용자들도 동일 PC에서 지속적으로 사용할 수 있습니다.
